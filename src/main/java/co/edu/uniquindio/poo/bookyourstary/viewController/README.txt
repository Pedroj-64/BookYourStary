/**
 * ================================================================
 * Clase: ViewController (JavaFX)
 * Propósito: Controlar la lógica de interfaz de una vista FXML
 * ================================================================
 *
 * Este controlador está asociado a una vista FXML específica.
 * Su responsabilidad es manejar los eventos de la interfaz gráfica
 * de usuario, coordinar la interacción con los servicios y actualizar
 * la vista en función del resultado de las operaciones.
 *
 * Funciones típicas:
 * - Capturar eventos (botones, campos de texto, etc.).
 * - Validar entradas básicas.
 * - Invocar métodos de la capa de servicios.
 * - Mostrar mensajes, actualizar listas o redirigir vistas.
 *
 * Importante: Este controlador no debe contener lógica de negocio 
 * ni acceder directamente a los datos. Todo eso debe delegarse a 
 * las capas correspondientes del sistema (service, repository).
 */
