/**

 * Carpeta: resources/
 * Propósito: Almacenar todos los recursos estáticos de la aplicación
 * ===============================================================
 *
 * Esta carpeta contiene archivos no Java que son utilizados por 
 * la aplicación durante su ejecución. Todos estos recursos son 
 * empaquetados junto con el proyecto y se pueden cargar fácilmente

 *
 * Contenidos típicos:
 * - Vistas FXML (interfaz gráfica de usuario).
 * - Imágenes de alojamientos, íconos, logos, etc.
 * - Archivos de estilos CSS personalizados para JavaFX.
 * - Archivos de configuración o textos multilenguaje (.properties).
 * - Recursos adicionales como fuentes, sonidos o datos JSON.
 *
 * Importante:
 * - Mantener organizada esta carpeta en subdirectorios si es necesario.
 * - Utilizar rutas relativas para acceder a los recursos desde FXML o Java.
